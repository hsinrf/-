هذا المشروع هو منصة مخصصة لتجميع ونشر نماذج الذكاء الاصطناعي، تم تصميمه ليكون واجهة سريعة وعصرية للمطورين والمستخدمين.
🚀 المميزات
تصميم عصري: واجهة مستخدم تعتمد على "Glassmorphism" وتأثيرات النيون.
متعدد المنصات: يدعم التشغيل كمتصفح ويب وكـ تطبيق أندرويد (APK).
سهولة التصفح: تنظيم ذكي لنماذج الذكاء الاصطناعي.
🛠️ تقنيات المشروع
Framework: React / Vite
Styling: Tailwind CSS
Platform: Capacitor (لتحويل الويب إلى تطبيق أندرويد)
📱 كيف تحصل على تطبيق الأندرويد (APK)؟
يمكنك الحصول على ملف التثبيت الخاص بالأندرويد عبر الخطوات التالية:
انتقل إلى تبويب Actions في هذا المستودع.
اختر آخر عملية بناء (Build) مكتملة بنجاح.
انزل إلى قسم Artifacts وقم بتحميل ملف app-debug.apk.
قم بتثبيت الملف على هاتفك واستمتع بالتطبيق!
