# dashboard-premium

لوحة تحكم Premium مع مساعد كود ومساعد نصوص مصغّرة مع تجربة مميزة لشاشات واسعة

مخطط معماري لتطبيق داشبورد حديث يركّز على الأداء العالي والتجاوب عبر شاشات واسعة. يتضمن مساعد كود مدمج وأدوات نصوص مصغّرة بميزة الوضع المصغّر مع ميزات متقدمة، بالإضافة إلى بنية مصادقة متقدمة وإدارة وصول RBAC وتكامل التحليلات. يراعي قابلية التوسع، الحماية، وتجربة مستخدم Premium عبر عناصر واجهة موحّدة وتحديث حي.

## 📱 تحويل المشروع إلى APK / Build Android APK

### الطريقة الأولى: محليًا (يحتاج JDK 17 + Android SDK)
```bash
npm install
npm run android:init     # أول مرة فقط — يضيف مجلد android/
npm run android:sync     # build + cap sync
npm run android:apk      # ينتج app-debug.apk
```
الناتج: `android/app/build/outputs/apk/debug/app-debug.apk`

### الطريقة الثانية: عبر GitHub Actions (بدون أي إعداد محلي)
ارفع المشروع إلى GitHub، وسيقوم `.github/workflows/android-apk.yml` ببناء الـ APK تلقائيًا وإتاحته كـ artifact.

### الطريقة الثالثة: PWABuilder
ارفع البناء إلى أي استضافة، ثم استخدم <https://www.pwabuilder.com/> لتوليد APK سريعًا.
